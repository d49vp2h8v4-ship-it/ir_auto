# IR Auto dependency jars

TrackAPI and UniversalModCore are included here for convenience (small files).
Immersive Railroading (106MB) is too large for git.

To get the IR jar:
1. Download ImmersiveRailroading-1.12.2-forge-1.10.0.jar from:
   https://www.curseforge.com/minecraft/mc-mods/immersive-railroading/files
2. Place it in this modir/ directory
